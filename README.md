## GSI - Gerenciador Simples de Inventário (Server)

### Sobre
O Projeto GSI tem por objetivo proporcionar o cadastro e gerenciamento de um inventário de ativos de maneira simplificada e direta.

### Tecnologias utilizadas:

• NestJS <br>
• Passport + JWT <br>
• Prisma + MySQL <br>
